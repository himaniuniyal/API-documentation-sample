## Base URL
https://jsonplaceholder.typicode.com

## Get All Users

### Endpoint
GET /users

### Description
Retrieves a list of all users.

### Request
No parameters are required for this request.

### Example Request
GET https://jsonplaceholder.typicode.com/users

### Example Response

[
  {
    "id": 1,
    "name": "Leanne Graham",
    "email": "leanne@example.com"
  }
]
